# shelby-onchain-dataset-test

Experimenting with Shelby decentralized storage for onchain analytics datasets.

## Use Case

This repository explores how Shelby storage could be used for:

- storing historical blockchain monitoring data
- storing DeFi protocol activity logs
- saving datasets for AI-assisted crypto analytics

## Example Dataset

A small example dataset simulating onchain monitoring logs is included.

Potential use cases:

- monitoring bots
- DeFi analytics
- AI trading signal datasets

## Status

Early experiment for testing Shelby as a storage layer for crypto analytics infrastructure.
